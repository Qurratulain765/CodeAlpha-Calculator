# CodeAlpha_Calculator 🧮

A modern, fully responsive, and feature-rich Calculator application built with HTML, CSS, and vanilla JavaScript. This project is part of my Frontend Development Internship at **CodeAlpha**, designed to demonstrate clean UI/UX and logic implementation.

---

## ✨ Key Features

- **Standard Arithmetic Operations:** Supports addition, subtraction, multiplication, and division with real-time result calculation[cite: 11, 12].
- **Keyboard Support:** Full compatibility with physical keyboard inputs, including numbers, operators, Enter (for result), and Backspace (for deletion)[cite: 12].
- **Responsive Design:** A mobile-first design approach using CSS Grid and media queries to ensure the calculator looks perfect on desktops, tablets, and mobile devices[cite: 13].
- **Advanced UI/UX:** 
    - Smooth animations like "pop" effects on results and pulse animations on the header[cite: 13].
    - Modern "dark mode" aesthetic with glassmorphism-inspired components and neon accents[cite: 13].
- **Robust Logic:** Includes error handling for invalid math expressions and automatic formatting for long numbers[cite: 12].

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure with modular button definitions.
- **CSS3:** Advanced styling utilizing Flexbox, CSS Grid, keyframe animations, and custom scrollbars[cite: 13].
- **JavaScript (Vanilla):** Dynamic DOM manipulation, event listeners for mouse/keyboard inputs, and secure `new Function()` math evaluation logic[cite: 12].
